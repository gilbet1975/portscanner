# Changelog

## v1.0.0 — Initial Release

### Added
- Multi-threaded port scanner
- Logging system with timestamped log files
- Diff analyzer for comparing scan results
- Clean project structure with modules
- README with logo, badges, installation & usage sections
- Phase-based roadmap

### Notes
This is the first stable release of the Portscanner Toolkit.  
The focus is on a clean foundation, modularity, and future extensibility.
